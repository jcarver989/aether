Aether is a modular, open-source AI coding agent toolkit written in Rust.


### Design principles

- **Empty by default** — Agents have no default system prompt or tools. You add what you need.
- **Any model** — Use any LLM you want, local or remote. Anthropic, OpenAI, OpenRouter, DeepSeek, Gemini, Z, Moonshot, LammaCpp, Ollama, and more are supported out of the box.
- **Any tools**— Agents get tools via [MCP](https://modelcontextprotocol.io/) severs. You can author tools in any language.
- **Any surface** — Run your agent in a TUI, IDE/editor (via ACP) or headless.
