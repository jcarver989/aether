import { Tabs, TabItem, Steps, FileTree, CardGrid, LinkCard } from '@astrojs/starlight/components';

<Steps>

1. **Install**

   <Tabs>
     <TabItem label="cargo install">
       ```bash
       cargo install aether-agent-cli
       ```
     </TabItem>
     <TabItem label="Build from source">
       ```bash
       git clone https://github.com/joshka/aether.git
       cd aether
       cargo build --release
       ```

       Binaries are written to `target/release/aether` and `target/release/wisp`.
     </TabItem>
   </Tabs>

   Requires **Rust 1.85+** — install via [rustup](https://rustup.rs) if you don't have it.

2. **Create your first agent**

   ```bash
   cd your-project
   aether agent new
   ```

   ```
   ✓ Created .aether/settings.json
   ✓ Created .aether/mcp.json
   ✓ Created .aether/SYSTEM.md
   ✓ Created AGENTS.md
   ```

   <FileTree>
     - .aether/
       - settings.json — [Agent definitions](/aether/configuration/agent-settings/)
       - mcp.json — [MCP server config](/aether/configuration/mcp-servers/)
       - SYSTEM.md — Base system prompt
     - AGENTS.md — [Project instructions](/aether/configuration/system-prompts/)
   </FileTree>

   :::note[Set up your LLM credentials]
   Most providers need an API key (e.g. `export ANTHROPIC_API_KEY=sk-ant-...`).
   See [LLM Providers](/aether/configuration/llm-providers/) for all supported providers and credential setup.
   :::

3. **Run it**

   **Interactive TUI**

   ```bash
   aether
   ```

   The default experience — a terminal interface for conversations with your agent.

   **IDE / editor (via ACP)**

   ```bash
   aether acp
   ```

   Exposes agents over [ACP](https://agentclientprotocol.org/) for [editor integration](/aether/running/editor-integration/).

   **Headless**

   ```bash
   aether headless "Explain the main function in this project"
   ```

   Run a single prompt and exit — ideal for scripts and CI. See [Headless CLI](/aether/running/headless/) for all flags and output formats.

</Steps>
