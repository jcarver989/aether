The default way to use Aether is through the Wisp terminal UI — a full-screen interactive interface for chatting with agents, reviewing diffs, and managing sessions.

```bash
aether
```

This launches the Wisp TUI with Aether as the backend agent. You can also run Wisp directly for more control:

```bash
wisp
wisp --agent "my-custom-agent serve"
```

## Features

- **Streaming responses** — See model output as it arrives
- **Inline diffs** — Review file changes in a two-pane diff viewer
- **Slash commands** — Trigger skills and commands with `/name`
- **Session persistence** — Resume previous conversations
- **Model selection** — Switch models on the fly
