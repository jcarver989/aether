title: Survey
description: Allow agents to ask structured questions via forms.
---

The `survey` server lets agents collect structured input from you during a workflow — confirmations, choices, text fields, numbers — without relying on free-form conversation.

## Tools

| Tool | Description |
|------|-------------|
| `ask_user` | Present a form to the user and collect structured responses |

### ask_user

Ask a structured question and collect the response via a form. The form fields are defined using a JSON Schema.

**`ask_user(message, schema)`**

- `message` — The question or prompt shown to the user
- `schema` — A JSON Schema object defining the form fields (must have `properties`)

Returns:

- `accepted` — `true` if the user submitted the form, `false` if they cancelled
- `data` — The structured response data (if accepted)

### Example

An agent might call `ask_user` to confirm a destructive operation:

```json
{
  "message": "This will delete 3 files. Continue?",
  "schema": {
    "type": "object",
    "properties": {
      "confirm": {
        "type": "boolean",
        "description": "Confirm deletion"
      }
    }
  }
}
```

Or to gather multiple inputs at once:

```json
{
  "message": "Configure the new service:",
  "schema": {
    "type": "object",
    "properties": {
      "name": {
        "type": "string",
        "description": "Service name"
      },
      "port": {
        "type": "number",
        "description": "Port number"
      },
      "public": {
        "type": "boolean",
        "description": "Expose publicly"
      }
    }
  }
}
```
