# Cost Tracking

**Priority:** 🔴 P0 - Critical
**Impact:** High
**Effort:** Medium
**Estimated LOC:** ~200

## Problem

There's currently no visibility into how much money eval runs cost. This creates several issues:

1. **Budget Blindness**: Organizations can't plan budgets for CI/CD eval runs
2. **No Optimization**: Can't identify which evals are expensive vs cheap
3. **Surprise Bills**: Users discover costs only when the API bill arrives
4. **No ROI Analysis**: Can't justify eval framework investment with cost data

### Real-World Scenario

```
Developer: "Let's run evals on every PR"
Manager: "How much will that cost?"
Developer: "Uh... no idea?"
Manager: "Then no."
```

## Solution

Track token usage and costs throughout the eval pipeline, then surface in results and UI.

### Data Model

Add cost tracking to storage models:

```rust
// In packages/crucible/src/storage/models.rs

/// Token usage and cost metrics for an eval
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct CostMetrics {
    /// Total input tokens sent to LLMs
    pub total_input_tokens: u64,

    /// Total output tokens generated by LLMs
    pub total_output_tokens: u64,

    /// Input tokens for agent LLM
    pub agent_input_tokens: u64,

    /// Output tokens for agent LLM
    pub agent_output_tokens: u64,

    /// Input tokens for judge LLM(s)
    pub judge_input_tokens: u64,

    /// Output tokens for judge LLM(s)
    pub judge_output_tokens: u64,

    /// Estimated cost in USD (based on model pricing)
    pub estimated_cost_usd: f64,

    /// Cost breakdown by component
    pub agent_cost_usd: f64,
    pub judge_cost_usd: f64,

    /// Model identifiers for pricing reference
    pub agent_model: String,
    pub judge_model: String,
}

impl CostMetrics {
    pub fn new(agent_model: String, judge_model: String) -> Self {
        Self {
            total_input_tokens: 0,
            total_output_tokens: 0,
            agent_input_tokens: 0,
            agent_output_tokens: 0,
            judge_input_tokens: 0,
            judge_output_tokens: 0,
            estimated_cost_usd: 0.0,
            agent_cost_usd: 0.0,
            judge_cost_usd: 0.0,
            agent_model,
            judge_model,
        }
    }

    pub fn add_agent_tokens(&mut self, input: u64, output: u64) {
        self.agent_input_tokens += input;
        self.agent_output_tokens += output;
        self.total_input_tokens += input;
        self.total_output_tokens += output;
    }

    pub fn add_judge_tokens(&mut self, input: u64, output: u64) {
        self.judge_input_tokens += input;
        self.judge_output_tokens += output;
        self.total_input_tokens += input;
        self.total_output_tokens += output;
    }

    /// Calculate cost based on model pricing
    /// Prices should be fetched from a pricing registry
    pub fn calculate_cost(&mut self, pricing: &PricingRegistry) {
        self.agent_cost_usd = pricing.calculate_cost(
            &self.agent_model,
            self.agent_input_tokens,
            self.agent_output_tokens,
        );

        self.judge_cost_usd = pricing.calculate_cost(
            &self.judge_model,
            self.judge_input_tokens,
            self.judge_output_tokens,
        );

        self.estimated_cost_usd = self.agent_cost_usd + self.judge_cost_usd;
    }
}

/// Registry of model pricing (per million tokens)
pub struct PricingRegistry {
    models: HashMap<String, ModelPricing>,
}

#[derive(Clone)]
pub struct ModelPricing {
    pub input_cost_per_million: f64,
    pub output_cost_per_million: f64,
}

impl Default for PricingRegistry {
    fn default() -> Self {
        Self::new()
    }
}

impl PricingRegistry {
    pub fn new() -> Self {
        let mut models = HashMap::new();

        // Anthropic pricing (as of Nov 2024)
        models.insert(
            "anthropic/claude-sonnet-4-5-20250929".to_string(),
            ModelPricing {
                input_cost_per_million: 3.0,
                output_cost_per_million: 15.0,
            },
        );

        models.insert(
            "anthropic/claude-3-5-haiku-20241022".to_string(),
            ModelPricing {
                input_cost_per_million: 1.0,
                output_cost_per_million: 5.0,
            },
        );

        // OpenAI pricing
        models.insert(
            "openai/gpt-4o".to_string(),
            ModelPricing {
                input_cost_per_million: 2.5,
                output_cost_per_million: 10.0,
            },
        );

        // Add more models as needed

        Self { models }
    }

    pub fn calculate_cost(&self, model: &str, input_tokens: u64, output_tokens: u64) -> f64 {
        let pricing = self.models.get(model).cloned().unwrap_or(ModelPricing {
            input_cost_per_million: 0.0,
            output_cost_per_million: 0.0,
        });

        let input_cost = (input_tokens as f64 / 1_000_000.0) * pricing.input_cost_per_million;
        let output_cost = (output_tokens as f64 / 1_000_000.0) * pricing.output_cost_per_million;

        input_cost + output_cost
    }
}
```

```rust
        &self,

        tx: Sender<AgentRunnerMessage>,
    ) -> Result<(), RunError> {
        // ... existing setup code ...

        let mut total_input_tokens = 0u64;
        let mut total_output_tokens = 0u64;

        loop {
            match stream.next().await {
                Some(Ok(agent_event)) => match agent_event {
                    AgentEvent::Text(text) => {
                        tx.send(AgentRunnerMessage::AgentText(text.clone())).await.ok();

                    // NEW: Track token usage from LLM responses
                    AgentEvent::Usage { input_tokens, output_tokens } => {
                        total_input_tokens += input_tokens as u64;
                        total_output_tokens += output_tokens as u64;

                        // Send token usage as a message
                        tx.send(AgentRunnerMessage::TokenUsage {
                            input_tokens: input_tokens as u64,
                            output_tokens: output_tokens as u64,
                        }).await.ok();
                    }

                    // ... rest of match arms
                }
            }
        }
    }
}
```

### Aggregate Costs in EvalRunner

```rust
// In packages/crucible/src/evals/runner.rs

async fn run_single_eval(...) -> Result<EvalResult, ...> {
    // ... existing code ...

    let mut cost_metrics = CostMetrics::new(
        agent_model_name.clone(),
        judge_model_name.clone(),
    );

    // Track agent tokens
    for msg in &messages {
        if let AgentRunnerMessage::TokenUsage { input_tokens, output_tokens } = msg {
            cost_metrics.add_agent_tokens(*input_tokens, *output_tokens);
        }
    }

    // Track judge tokens (modify assert_llm_judge to return token usage)
    for assertion in &eval.assertions {
        if let EvalAssertion::LLMJudge { .. } = assertion {
            let (result, judge_tokens) = assert_llm_judge_with_tracking(...).await;
            cost_metrics.add_judge_tokens(judge_tokens.input, judge_tokens.output);
        }
    }

    // Calculate final costs
    let pricing_registry = PricingRegistry::new();
    cost_metrics.calculate_cost(&pricing_registry);

    Ok(EvalResult::Completed {
        id,
        eval_name,
        passed,
        assertions,
        agent_diff,
        reference_diff,
        cost_metrics: Some(cost_metrics),
    })
}
```

```rust
println!("\n{}", "Cost Summary".bold());

let total_cost: f64 = results
    .iter()
    .filter_map(|r| {
        if let EvalResult::Completed { cost_metrics, .. } = r {

            None
        }

    .iter()
    .filter_map(|r| {

        } else {
            None
        }
    })

println!("Total tokens: {}", total_tokens.to_string().bright_cyan());
```

## Error Handling

Following the error-handling best practice, define specific error types for cost tracking:

```rust
#[derive(Debug)]
pub enum CostTrackingError {
    UnknownModel { model: String },
    InvalidTokenCount { reason: String },
    PricingDataMissing,
}

impl std::fmt::Display for CostTrackingError {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            Self::UnknownModel { model } => write!(f, "Unknown model: {}", model),
            Self::InvalidTokenCount { reason } => write!(f, "Invalid token count: {}", reason),
            Self::PricingDataMissing => write!(f, "Pricing data is missing"),
        }
    }
}

impl std::error::Error for CostTrackingError {}

pub type Result<T> = std::result::Result<T, CostTrackingError>;
```

## Testing Strategy

Following the testing-fakes.md best practice (use "Fake", never "Mock"):

Create a `FakePricingRegistry` for deterministic testing:

```rust
/// Fake pricing registry for testing cost calculations
pub struct FakePricingRegistry {
    fixed_input_cost: f64,
    fixed_output_cost: f64,
}

impl FakePricingRegistry {
    pub fn new(input_cost_per_million: f64, output_cost_per_million: f64) -> Self {
        Self {
            fixed_input_cost: input_cost_per_million,
            fixed_output_cost: output_cost_per_million,
        }
    }

    pub fn calculate_cost(&self, _model: &str, input_tokens: u64, output_tokens: u64) -> f64 {
        let input_cost = (input_tokens as f64 / 1_000_000.0) * self.fixed_input_cost;
        let output_cost = (output_tokens as f64 / 1_000_000.0) * self.fixed_output_cost;
        input_cost + output_cost
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_cost_calculation() {
        let pricing = FakePricingRegistry::new(3.0, 15.0);

        // 1M input tokens + 1M output tokens
        let cost = pricing.calculate_cost("test-model", 1_000_000, 1_000_000);
        assert_eq!(cost, 18.0); // 3.0 + 15.0
    }

    #[test]
    fn test_cost_metrics_aggregation() {
        let mut metrics = CostMetrics::new("agent-model".to_string(), "judge-model".to_string());

        metrics.add_agent_tokens(100_000, 50_000);
        metrics.add_judge_tokens(10_000, 5_000);

        assert_eq!(metrics.total_input_tokens, 110_000);
        assert_eq!(metrics.total_output_tokens, 55_000);
    }
}
```
