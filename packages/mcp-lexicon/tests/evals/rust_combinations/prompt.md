# Write a Rust program to compute combinations

Write a Rust program that computes all combinations of a given set of characters. The program should take a string of characters and generate all possible combinations of those characters, including combinations of different lengths.

For example, given the characters "abc", the program should output:
- Single characters: a, b, c
- Two-character combinations: ab, ac, bc
- Three-character combination: abc

The program should:
1. Accept input as a command-line argument or hardcoded string
2. Generate all possible combinations (not permutations - order doesn't matter)
3. Print each combination on a separate line
4. Handle empty input gracefully
