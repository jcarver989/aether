# Create a complex todo list with all states

You are working on building a new web application called "TaskMaster" with multiple components. Please create a comprehensive todo list for this project that includes:

1. At least 3 tasks that are currently pending (not started yet)
2. At least 2 tasks that are currently in progress (being worked on right now)  
3. At least 2 tasks that have been completed
4. Each task should have both the imperative content form (e.g., "Set up database schema") and the active form (e.g., "Setting up database schema")
5. Tasks should be realistic for building a web application (frontend, backend, database, testing, deployment, etc.)
6. Make sure to follow the todo write tool requirements exactly for the task structure

Start by marking one of the pending tasks as in_progress, then create the full list.
